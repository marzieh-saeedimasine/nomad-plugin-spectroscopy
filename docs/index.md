# Welcome to NOMAD Spectroscopy Plugin

The NOMAD Spectroscopy Plugin extends [NOMAD](https://nomad-lab.eu/) with comprehensive support for parsing and managing spectroscopy data, including:

- **IR Spectroscopy** - Infrared spectroscopy data
- **Raman Spectroscopy** - Raman scattering data
- **UV-Vis Spectroscopy** - Ultraviolet-Visible spectroscopy

## Key Features

✨ **Easy Integration** - Seamlessly integrates with NOMAD's data management platform

📊 **Flexible Parsing** - Automatically detects and parses spectroscopy CSV files

🔧 **Customizable** - Flexible column name matching for various data formats

📈 **Searchable** - Use NOMAD's powerful search and analytics on spectroscopy data

🚀 **Well-Documented** - Comprehensive documentation and examples

## Quick Start

Get started with the plugin:

```bash
# Install
pip install nomad-plugin-spectroscopy

# Use in your NOMAD instance
# Upload CSV files and let the plugin handle the parsing
```
